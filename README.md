# htmx-ts-starter

To install dependencies:

```bash
bun i
```

To run:

```bash
bun dev
```

# Features

- [x] [htmx](https://htmx.org/)
- [x] [tailwindcss](https://tailwindcss.com/)
- [x] [elysiajs](https://elysiajs.com/)
- [x] [daisyui](https://daisyui.com/)
